## Development 

```bash
# Clone the repository 
git clone https://github.com/Pacerino/pr0music_frontend

# Enter into the directory
cd pr0music_frontend

# Install the dependencies
yarn

# Copy the .env
cp .env.example .env

# Fill out the .env
nano .env

# Start the Server
yarn dev
```
